# 면접 대비 학습 허브

개발자 면접 대비 마크다운 학습 자료를 하나의 웹사이트로 묶어 보는 정적 사이트입니다.
서버·빌드 도구 없이 **`index.html` 을 더블클릭**하면 바로 열립니다.

## 구성

```
site/
├─ index.html              ← 허브(대문). 주제별 카드 자동 배치
├─ doc_*.html              ← 문서별 페이지 (md 하나당 하나)
├─ assets/
│   ├─ style.css           ← 공통 스타일 (라이트/다크 테마)
│   └─ app.js              ← 마크다운 렌더러 + 검색/TOC/다크모드/아코디언
├─ md/                     ← 원본 마크다운 (여기에 파일을 넣습니다)
└─ build.py                ← 사이트 생성기
```

## 기능

- **무손실 렌더링** — 원문 마크다운을 그대로 페이지에 심어 브라우저에서 변환. 요약·생략 없음.
- **전체 검색** — 상단 검색창(단축키 `Ctrl+/`). 모든 문서 대상, 결과 클릭 시 해당 문서로 이동.
- **사이드바 목차(TOC)** — `##`/`###` 자동 수집, 현재 위치 하이라이트.
- **확인질문 아코디언** — `checkquestions` 문서는 질문을 클릭하면 펼쳐지는 자가 점검 형태.
- **다크모드 토글** — 우상단 버튼. 시스템 설정을 기본값으로 따름.
- **코드 복사 버튼**, **표 가로 스크롤**, **모바일 반응형**.
- 코드 문법 강조는 인터넷이 있으면 highlight.js(CDN)를 쓰고, 없으면 자동으로 생략(핵심 동작엔 영향 없음).

## 새 마크다운(주제) 추가하는 법 — 3단계

1. **파일 넣기**: `md/` 폴더에 규칙대로 파일을 넣습니다.
   `{번호}_{주제}_{분류}.md` 예) `10_db_basic.md`, `13_db_checkquestions.md`
   분류: `basic`(기본) · `supplement`(부가) · `essentials`(기초) · `checkquestions`(확인질문) · `practical`(실전)
2. **주제 이름 등록**(새 주제일 때만): `build.py` 상단 `TOPIC_LABEL` 에 한 줄 추가 — `"db": "데이터베이스",`
3. **재생성**: 터미널에서 `python3 build.py` 실행 → 카드와 페이지가 자동 생성됩니다.

> 공통 문서(`0_answering_technique.md` 처럼 주제 없이 `{번호}_{분류}`)는 허브 최상단 배너로 배치됩니다.
> 새 분류를 만들려면 `build.py` 의 `KIND_META` 에 라벨·설명·색상을 추가하세요.

## 다시 빌드하기

```bash
cd site
python3 build.py
```

Python 3 만 있으면 되고 외부 라이브러리는 필요 없습니다.
